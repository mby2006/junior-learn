## 项目概览扫描
- 项目：StudyBuddy —— 初中生自学辅助全栈系统
- 技术栈：Python FastAPI + Alpine.js + Tailwind（单文件 SPA，无构建步骤）
- 端口：8001，入口：src/api/run_server.py
- 核心功能：
  1. 作业批改：OcrAgent → GradeAgent → KnowPointAgent → ExamTagAgent（LlamaParse OCR）
  2. 题目讲解答疑：ExplainAgent，多轮对话，教材 RAG 检索增强
  3. 智能出题：Retrieve → Generate → Analyze，支持知识库自定义模式和模拟试卷模式，WebSocket 流式推送
  4. 错题本：CRUD + 统计，本地 JSON 存储
  5. 学情记忆/画像：MemoryAgent，可接 EverMemOS
  6. 教材管理：9 科 PDF 知识库（数理化英语文史生政地）已建索引
  7. 阅读材料管理（仅在线阅读，不参与 RAG）
  8. 历史记录、用户设置、健康检查
  9. 恐龙快跑小游戏（iframe 内嵌，postMessage 通信）
- LLM：OpenAI 兼容接口，支持 OpenAI/Anthropic/Gemini/Kimi
- RAG：raganything 引擎，embedding 用 text-embedding-3-small
- 部署：Uvicorn 本地 / Docker Compose / systemd 生产
